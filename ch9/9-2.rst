9.2 析构函数
==============

这里只是简单介绍下析构函数，一般来说，当 main函数中执行完一遍函数的时候，最后才执行析构函数。

先看下例子

.. code-block:: c++ 

    class Time
    {
    public:
       Time()       //构造函数的一般形式
       {
         hour = 0;
         min  = 0;
         sec  = 0;
       } 
       ～Time()
       {
          cout << "这是析构函数"<<endl;
       }

    private:
       int hour;
       int min;
       int sec;
    };            //别忘记写分号

所以析构函数的形式就是 "~"符号 + 类名

 **~ + 类名**

 {
	对象最后一次执行的操作; //比如上面的例子是输出一句话
 }
